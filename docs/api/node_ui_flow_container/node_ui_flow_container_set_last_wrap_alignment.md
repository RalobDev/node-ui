# NodeUI.FlowContainer:setLastWrapAlignment()

Define o alinhamento dos filhos da última linha ou coluna.


## Sinopse

```lua
NodeUI.FlowContainer:setLastWrapAlignment(alignment)
```

## Argumentos
- **`NodeUI.FlowContainer.LastWrapAlignmentMode` alignment** <br>
Alinhamento dos filhos.

## Retornos
Nenhum.
