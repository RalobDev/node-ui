# NodeUI.FlowContainer:getSeparation()

Retorna a separação entre os filhos.


## Sinopse

```lua
separation = NodeUI.FlowContainer:getSeparation(axis)
```

## Argumentos
- **`NodeUI.Control.Axis` axis** <br>
Eixo da separação.

## Retornos
- `number` **separation** <br>
Separação em pixels.
