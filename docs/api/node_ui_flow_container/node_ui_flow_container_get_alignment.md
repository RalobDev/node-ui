# NodeUI.FlowContainer:getAlignment()

Retorna o alinhamento dos filhos.


## Sinopse

```lua
alignment = NodeUI.FlowContainer:getAlignment()
```

## Argumentos
Nada.

## Retornos
- `NodeUI.Control.AlignmentMode` **alignment** <br>
Alinhamento dos filhos.
