# NodeUI.FlowContainer:new()

Cria um novo **FlowContainer**.


## Sinopse

```lua
FlowContainer = NodeUI.FlowContainer:new(x, y, width, height)
```

## Argumentos
- **`number` x** <br>
Posição horizontal.
- **`number` y** <br>
Posição vertical.
- **`number` width** <br>
Comprimento em pixels.
- **`number` height** <br>
Altura em pixels.

## Retornos
- [`NodeUI.FlowContainer`](../node_ui_flow_container/node_ui_flow_container.md) **FlowContainer** <br>
Novo **FlowContainer**.
