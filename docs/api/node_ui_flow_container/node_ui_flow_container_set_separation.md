# NodeUI.FlowContainer:setSeparation()

Define a separação entre os filhos.


## Sinopse

```lua
NodeUI.FlowContainer:setSeparation(axis, separation)
```

## Argumentos
- **`NodeUI.Control.Axis` axis** <br>
Eixo da separação.
- **`number` separation** <br>
Separação em pixels.

## Retornos
Nenhum.
