# NodeUI.FlowContainer:setVertical()

Define se a organização dos filhos é vertical ou não. Por padrão `enabled` é `true`.


## Sinopse

```lua
NodeUI.FlowContainer:setVertical(enabled?)
```

## Argumentos
- **`boolean` enabled?** <br>
Se é para organizar verticalmente.

## Retornos
Nenhum.
