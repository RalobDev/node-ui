# NodeUI.AspectRatioContainer:getStretchMode()

Retorna a maneira como escalona os filhos.


## Sinopse

```lua
stretch_mode = NodeUI.AspectRatioContainer:getStretchMode()
```

## Argumentos
Nada.

## Retornos
- `NodeUI.AspectRatioContainer.StretchMode` **stretch_mode** <br>
Modo de escalonamento.
