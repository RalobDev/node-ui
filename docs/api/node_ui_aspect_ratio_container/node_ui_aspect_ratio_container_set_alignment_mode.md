# NodeUI.AspectRatioContainer:setAlignmentMode()

Define o `AlignmentMode` aplicado aos filhos.


## Sinopse

```lua
NodeUI.AspectRatioContainer:setAlignmentMode(axis, alignment_mode)
```

## Argumentos
- **`NodeUI.Control.Axis` axis**
Eixo do alinhamento.
- **`NodeUI.Control.AlignmentMode` alignment_mode**
Modo de alinhamento.

## Retornos
Nenhum.
