# NodeUI.AspectRatioContainer:getAlignmentMode()

Retorna o `AlignmentMode` aplicado aos filhos.


## Sinopse

```lua
alignment_mode = NodeUI.AspectRatioContainer:getAlignmentMode(axis)
```

## Argumentos
- **`NodeUI.Control.Axis` axis**
Eixo de alinhamento.

## Retornos
- `NodeUI.Control.AlignmentMode` **alignment_mode**
Modo de alinhamento.
