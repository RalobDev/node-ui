# NodeUI.AspectRatioContainer:connect()

Cria uma conexão em determinado sinal do [Control](../node_ui_control/node_ui_control.md).


## Sinopse

```lua
NodeUI.AspectRatioContainer:connect(signal, owner, method)
```

## Argumentos
- **`NodeUI.Control.Signals` signal**
Nome do sinal.
- **`table` owner**
Objeto dono do método.
- **`string` method**
Nome do método chamado ao sinal ser emitido.

## Retornos
Nenhum.
