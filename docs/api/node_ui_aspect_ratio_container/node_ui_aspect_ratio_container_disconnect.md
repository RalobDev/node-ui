# NodeUI.AspectRatioContainer:disconnect()

Remove a conexão de um sinal do [Control](../node_ui_control/node_ui_control.md).


## Sinopse

```lua
NodeUI.AspectRatioContainer:disconnect(signal, method)
```

## Argumentos
- **`NodeUI.Control.Signals` signal**
Nome do sinal.
- **`string` method**
Nome do método chamado ao sinal ser emitido.

## Retornos
Nenhum.
