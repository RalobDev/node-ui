# NodeUI.Control:getMinimumDimensions()

Retorna a dimensão mínima do **Control**.


## Sinopse

```lua
width, height = NodeUI.Control:getMinimumDimensions()
```

## Argumentos
Nada.

## Retornos
- `number` **width** <br>
Comprimento mínimo do **Control**.
- `number` **height** <br>
Altura mínima do **Control**.
