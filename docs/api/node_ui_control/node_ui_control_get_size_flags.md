# NodeUI.Control:getSizeFlags()

Retorna a size flags do `axis`.


## Sinopse

```lua
size_flags = NodeUI.Control:getSizeFlags(axis)
```

## Argumentos
- **`NodeUI.Control.Axis` axis** <br>
Eixo da size flags.

## Retornos
- `NodeUI.Control.SizeFlags` **size_flags** <br>
Size flags aplicada ao `axis`.
