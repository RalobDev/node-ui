# NodeUI.Control:setSizeFlags()

Define a size flags do `axis`.


## Sinopse

```lua
NodeUI.Control:setSizeFlags(axis, size_flags)
```

## Argumentos
- **`NodeUI.Control.Axis` axis** <br>
Eixo da size flags.
- **`NodeUI.Control.SizeFlags` size_flags** <br>
Size flags aplicada ao `axis`.

## Retornos
Nenhum.
