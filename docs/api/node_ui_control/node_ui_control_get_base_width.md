# NodeUI.Control:getBaseWidth()

Retorna o comprimento base do **Control**. É o comprimento definido ao criar o **Control** e ao chamar
`Control:setWidth()`.


## Sinopse

```lua
width = NodeUI.Control:getBaseWidth()
```

## Argumentos
Nada.

## Retornos
- `number` **width** <br>
Comprimento base.
