# NodeUI.Control:setClipContent()

Define o recorte de conteúdo do **Control**. Se `true`, clipa o desenho dos filhos à área do **Control**.
Por padrão ativa o recorte de conteúdo.


## Sinopse

```lua
NodeUI.Control:setClipContent(enabled?)
```

## Argumentos
- **`boolean` enabled?** <br>


## Retornos
Nenhum.
