# NodeUI.Control:getChildren()

Retorna uma tabela com todos os filhos do **Control**.


## Sinopse

```lua
children = NodeUI.Control:getChildren(include_internal?)
```

## Argumentos
- **`boolean` include_internal?**
Se `true`, retorna os filhos internos também.

## Retornos
- `NodeUI.Control[]` **children**
Filhos do **Control**.
