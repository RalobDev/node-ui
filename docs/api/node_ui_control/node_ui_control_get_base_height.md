# NodeUI.Control:getBaseHeight()

Retorna a altura base do **Control**. É a altura definida ao criar o **Control** e ao chamar
`Control:setHeight()`.


## Sinopse

```lua
height = NodeUI.Control:getBaseHeight()
```

## Argumentos
Nada.

## Retornos
- `number` **height** <br>
Altura base.
