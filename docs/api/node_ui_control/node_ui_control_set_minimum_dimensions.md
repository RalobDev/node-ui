# NodeUI.Control:setMinimumDimensions()

Define a dimensão mínima do **Control**.


## Sinopse

```lua
NodeUI.Control:setMinimumDimensions(width, height)
```

## Argumentos
- **`number` width**
Novo comprimento mínimo.
- **`number` height**
Nova altura mínima.

## Retornos
Nenhum.
