# NodeUI.Control:getBaseDimensions()

Retorna a dimensão base do **Control**. É a dimensão definida ao criar o **Control** e ao chamar
`Control:setDimensions()`.


## Sinopse

```lua
width, height = NodeUI.Control:getBaseDimensions()
```

## Argumentos
Nada.

## Retornos
- `number` **width** <br>
Comprimento base.
- `number` **height** <br>
Altura base.
