# NodeUI.Control:disconnect()

Remove a conexão de um sinal do **Control**.


## Sinopse

```lua
NodeUI.Control:disconnect(signal, method)
```

## Argumentos
- **`NodeUI.Control.Signals` signal** <br>
Nome do sinal.
- **`string` method** <br>
Nome do método chamado ao sinal ser emitido.

## Retornos
Nenhum.
