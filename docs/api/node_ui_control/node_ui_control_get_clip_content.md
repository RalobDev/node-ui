# NodeUI.Control:getClipContent()

Retorna se o recorte de conteúdo do **Control** está ativado.


## Sinopse

```lua
clip_content = NodeUI.Control:getClipContent()
```

## Argumentos
Nada.

## Retornos
- `boolean` **clip_content** <br>
Se o recorte de conteúdo está ativo.
