# NodeUI.Control:isQueuedForDeletion()

Retorna se o **Control** está na fila de deleção.


## Sinopse

```lua
deletion = NodeUI.Control:isQueuedForDeletion()
```

## Argumentos
Nada.

## Retornos
- `boolean` **deletion**
Se `true`, o **Control** está na fila de deleção.
