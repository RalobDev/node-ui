# NodeUI.Control:getDimensions()

Retorna a dimensão do **Control**.


## Sinopse

```lua
width, height = NodeUI.Control:getDimensions()
```

## Argumentos
Nada.

## Retornos
- `number` **width**
Comprimento do **Control**.
- `number` **height**
Altura do **Control**.
