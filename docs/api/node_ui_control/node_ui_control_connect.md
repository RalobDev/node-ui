# NodeUI.Control:connect()

Cria uma conexão em determinado sinal do **Control**.


## Sinopse

```lua
NodeUI.Control:connect(signal, owner, method)
```

## Argumentos
- **`NodeUI.Control.Signals` signal**
Nome do sinal.
- **`table` owner**
Objeto dono do método.
- **`string` method**
Nome do método chamado ao sinal ser emitido.

## Retornos
Nenhum.
