# NodeUI.BoxContainer:getAlignment()

Retorna o alinhamento dos filhos.


## Sinopse

```lua
alignment = NodeUI.BoxContainer:getAlignment()
```

## Argumentos
Nada.

## Retornos
- `NodeUI.Control.AlignmentMode` **alignment** <br>
Alinhamento dos filhos.
