# NodeUI.BoxContainer

Organiza os filhos horizontalmente ou verticalmente.


## Métodos

| Nome | Descrição | Retornos |
| ---- | --------- | -------- |
[connect](node_ui_box_container_connect.md) | Cria uma conexão em determinado sinal do [Control](../node_ui_control/node_ui_control.md). | `nil`
[disconnect](node_ui_box_container_disconnect.md) | Remove a conexão de um sinal do [Control](../node_ui_control/node_ui_control.md). | `nil`
[getAlignment](node_ui_box_container_get_alignment.md) | Retorna o alinhamento dos filhos. | `NodeUI.Control.AlignmentMode`
[getSeparation](node_ui_box_container_get_separation.md) | Retorna a separação entre os filhos. | `number`
[getVertical](node_ui_box_container_get_vertical.md) | Retorna se a organização dos filhos é vertical ou não. | `boolean`
[new](node_ui_box_container_new.md) | Cria um novo **BoxContainer**. | [`NodeUI.BoxContainer`](../node_ui_box_container/node_ui_box_container.md)
[setAlignment](node_ui_box_container_set_alignment.md) | Define o alinhamento dos filhos. | `nil`
[setSeparation](node_ui_box_container_set_separation.md) | Define a separação entre os filhos. | `nil`
[setVertical](node_ui_box_container_set_vertical.md) | Define se a organização dos filhos é vertical ou não. Por padrão `enabled` é `true`. | `nil`
