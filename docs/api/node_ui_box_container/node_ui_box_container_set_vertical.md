# NodeUI.BoxContainer:setVertical()

Define se a organização dos filhos é vertical ou não. Por padrão `enabled` é `true`.


## Sinopse

```lua
NodeUI.BoxContainer:setVertical(enabled?)
```

## Argumentos
- **`boolean` enabled?** <br>
Se é para organizar verticalmente.

## Retornos
Nenhum.
