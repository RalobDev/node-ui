# NodeUI.BoxContainer:getVertical()

Retorna se a organização dos filhos é vertical ou não.


## Sinopse

```lua
enabled = NodeUI.BoxContainer:getVertical()
```

## Argumentos
Nada.

## Retornos
- `boolean` **enabled** <br>
Se é organizado verticalmente.
