# NodeUI.setBaseDimensions()

Define a dimensão base dos [Control](../node_ui_control/node_ui_control.md) na raiz.


## Sinopse

```lua
NodeUI:setBaseDimensions(width, height)
```

## Argumentos
- **`number` width**
Novo comprimento.
- **`number` height**
Nova altura.

## Retornos
Nenhum.
