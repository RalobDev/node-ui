# NodeUI.getBasePosition()

Retorna a posição base dos [Control](../node_ui_control/node_ui_control.md) na raiz.


## Sinopse

```lua
x, y = NodeUI:getBasePosition()
```

## Argumentos
Nada.

## Retornos
- `number` **x**
Posição base x.
- `number` **y**
Posição base y.
