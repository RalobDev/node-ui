# NodeUI.getBaseDimensions()

Retorna a dimensão base dos [Control](../node_ui_control/node_ui_control.md) na raiz.


## Sinopse

```lua
width, height = NodeUI:getBaseDimensions()
```

## Argumentos
Nada.

## Retornos
- `number` **width**
Comprimento base.
- `number` **height**
Altura base.
