# NodeUI.keyreleased()

Lida com o soltar de teclas.


## Sinopse

```lua
NodeUI:keyreleased(key, scancode)
```

## Argumentos
- **`love.KeyConstant` key**
Caractere da tecla pressionada.
- **`love.Scancode` scancode**
O scancode que representa a tecla pressionada.

## Retornos
Nenhum.
