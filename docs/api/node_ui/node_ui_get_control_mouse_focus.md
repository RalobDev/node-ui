# NodeUI.getControlMouseFocus()

Retorna o [Control](../node_ui_control/node_ui_control.md) que possui o foco do mouse. Se nenhum possuir, retorna `nil`.


## Sinopse

```lua
mouse_focus_control = NodeUI:getControlMouseFocus()
```

## Argumentos
Nada.

## Retornos
- `NodeUI.Control?` **mouse_focus_control** <br>
**Control** que possui o foco do mouse.
