# NodeUI.mousepressed()

Lida com o pressionar de um botão do mouse.


## Sinopse

```lua
NodeUI:mousepressed(x, y, button, istouch, presses)
```

## Argumentos
- **`number` x** <br>
Posição x do mouse, em pixels.
- **`number` y** <br>
Posição y do mouse, em pixels.
- **`number` button** <br>
O index do botão que foi pressionado.
- **`boolean` istouch** <br>
`true` se o pressionar do botão do mouse é originado de uma touchscreen.
- **`number` presses** <br>
O número de pressionamentos.

## Retornos
Nenhum.
