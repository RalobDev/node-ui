# NodeUI.mousereleased()

Lida com o soltar de um botão do mouse.


## Sinopse

```lua
NodeUI:mousereleased(x, y, button, istouch, presses)
```

## Argumentos
- **`number` x**
Posição x do mouse, em pixels.
- **`number` y**
Posição y do mouse, em pixels.
- **`number` button**
O index do botão que foi solto.
- **`boolean` istouch**
`true` se o soltar do botão do mouse é originado de uma touchscreen.
- **`number` presses**
O número de pressionamentos.

## Retornos
Nenhum.
