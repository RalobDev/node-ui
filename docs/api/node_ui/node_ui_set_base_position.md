# NodeUI.setBasePosition()

Define a posição base dos [Control](../node_ui_control/node_ui_control.md).


## Sinopse

```lua
NodeUI:setBasePosition(x, y)
```

## Argumentos
- **`number` x** <br>
Nova posição x.
- **`number` y** <br>
Nova posição y.

## Retornos
Nenhum.
