# NodeUI.sendBaseMousePosition()

Envia para o **NodeUI** qual deve ser a posição do mouse usada pelos nós.


## Sinopse

```lua
NodeUI:sendBaseMousePosition(x, y)
```

## Argumentos
- **`number` x** <br>
Posição x do mouse.
- **`number` y** <br>
Posição y do mouse.

## Retornos
Nenhum.
