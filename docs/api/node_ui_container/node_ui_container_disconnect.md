# NodeUI.Container:disconnect()

Desconecta o `method` do `signal`.


## Sinopse

```lua
NodeUI.Container:disconnect(signal, method)
```

## Argumentos
- **`NodeUI.Control.Signals` signal**
Nome do sinal.
- **`string` method**
Nome do método chamado ao sinal ser emitido.

## Retornos
Nenhum.
