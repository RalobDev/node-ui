# NodeUI.Container:connect()

Cria uma conexão em determinado sinal do [Control](../node_ui_control/node_ui_control.md).


## Sinopse

```lua
NodeUI.Container:connect(signal, owner, method)
```

## Argumentos
- **`NodeUI.Control.Signals` signal** <br>
Nome do sinal.
- **`table` owner** <br>
Objeto dono do método.
- **`string` method** <br>
Nome do método chamado ao sinal ser emitido.

## Retornos
Nenhum.
