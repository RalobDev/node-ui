# NodeUI.Container:new()

Cria um novo **Container**.


## Sinopse

```lua
Container = NodeUI.Container:new(x, y, width, height)
```

## Argumentos
- **`number` x**
Posição horizontal.
- **`number` y**
Posição vertical.
- **`number` width**
Comprimento em pixels.
- **`number` height**
Altura em pixels.

## Retornos
- [`NodeUI.Container`](../node_ui_container/node_ui_container.md) **Container**
Novo **Container**.
