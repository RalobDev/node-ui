# NodeUI.MarginContainer

O **MarginContainer** permite criar uma margem ao redor dos filhos.


## Métodos

| Nome | Descrição | Retornos |
| ---- | --------- | -------- |
[_onDrawDebug](node_ui_margin_container__on_draw_debug.md) |  | `nil`
[connect](node_ui_margin_container_connect.md) | Cria uma conexão em determinado sinal do [Control](../node_ui_control/node_ui_control.md). | `nil`
[disconnect](node_ui_margin_container_disconnect.md) | Desconecta o `method` do `signal`. | `nil`
[getMargin](node_ui_margin_container_get_margin.md) | Define a margem de um lado do **MarginContainer**. | `number`
[new](node_ui_margin_container_new.md) | Cria um novo **MarginContainer**. | [`NodeUI.MarginContainer`](../node_ui_margin_container/node_ui_margin_container.md)
[setMargin](node_ui_margin_container_set_margin.md) | Define a margem de um lado do **MarginContainer**. | `nil`
