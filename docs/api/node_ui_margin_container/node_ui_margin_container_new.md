# NodeUI.MarginContainer:new()

Cria um novo **MarginContainer**.


## Sinopse

```lua
MarginContainer = NodeUI.MarginContainer:new(x, y, width, height)
```

## Argumentos
- **`number` x** <br>
Posição horizontal.
- **`number` y** <br>
Posição vertical.
- **`number` width** <br>
Comprimento em pixels.
- **`number` height** <br>
Altura em pixels.

## Retornos
- [`NodeUI.MarginContainer`](../node_ui_margin_container/node_ui_margin_container.md) **MarginContainer** <br>
Novo **MarginContainer**.
