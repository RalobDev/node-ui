# NodeUI.MarginContainer:disconnect()

Desconecta o `method` do `signal`.


## Sinopse

```lua
NodeUI.MarginContainer:disconnect(signal, method, owner)
```

## Argumentos
- **`NodeUI.MarginContainer.Signals` signal** <br>
Nome do sinal.
- **`string|function` method** <br>
Nome do método ou método chamado ao sinal ser emitido.
- **`table?` owner** <br>
Objeto dono do método.

## Retornos
Nenhum.
