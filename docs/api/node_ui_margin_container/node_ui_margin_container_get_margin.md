# NodeUI.MarginContainer:getMargin()

Define a margem de um lado do **MarginContainer**.


## Sinopse

```lua
margin = NodeUI.MarginContainer:getMargin(side)
```

## Argumentos
- **`NodeUI.Control.Side` side** <br>
Lado da margem.

## Retornos
- `number` **margin** <br>
Margem do lado.
