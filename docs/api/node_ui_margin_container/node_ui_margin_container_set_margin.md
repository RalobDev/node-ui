# NodeUI.MarginContainer:setMargin()

Define a margem de um lado do **MarginContainer**.


## Sinopse

```lua
NodeUI.MarginContainer:setMargin(side, margin)
```

## Argumentos
- **`NodeUI.Control.Side` side** <br>
Lado da margem.
- **`number` margin** <br>
Margem do lado.

## Retornos
Nenhum.
