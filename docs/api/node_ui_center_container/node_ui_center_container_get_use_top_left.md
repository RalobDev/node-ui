# NodeUI.CenterContainer:getUseTopLeft()

Retorna se o **CenterContainer** centraliza os seus filhos no topo esquerdo ou não.


## Sinopse

```lua
use_top_left = NodeUI.CenterContainer:getUseTopLeft()
```

## Argumentos
Nada.

## Retornos
- `boolean` **use_top_left** <br>
Se a centralização no topo esquerdo está ativada.
