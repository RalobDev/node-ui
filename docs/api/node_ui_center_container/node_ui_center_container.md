# NodeUI.CenterContainer

O **CenterContainer** centraliza seus filhos.
 
## Descrição
 
A centralização dos filhos pode acontecer no centro do próprio **CenterContainer** ou no seu canto superior esquerdo.
Para alterar entre os dois modos use `CenterContainer:setUseTopLeft()`.


## Métodos

| Nome | Descrição | Retornos |
| ---- | --------- | -------- |
[getUseTopLeft](node_ui_center_container_get_use_top_left.md) | Retorna se o **CenterContainer** centraliza os seus filhos no topo esquerdo ou não. | `boolean`
[new](node_ui_center_container_new.md) | Cria um novo **CenterContainer**. | [`NodeUI.CenterContainer`](../node_ui_center_container/node_ui_center_container.md)
[setUseTopLeft](node_ui_center_container_set_use_top_left.md) | Define se o **CenterContainer** deve centralizar os seus filhos no topo esquerdo ou não. Por padrão ativa a centralização no topo esquerdo. | `nil`
