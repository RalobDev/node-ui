# NodeUI.GridContainer:disconnect()

Remove a conexão de um sinal do [Control](../node_ui_control/node_ui_control.md).


## Sinopse

```lua
NodeUI.GridContainer:disconnect(signal, method, owner)
```

## Argumentos
- **`NodeUI.GridContainer.Signals` signal** <br>
Nome do sinal.
- **`string|function` method** <br>
Nome do método ou método chamado ao sinal ser emitido.
- **`table?` owner** <br>
Objeto dono do método.

## Retornos
Nenhum.
