# NodeUI.GridContainer:new()

Cria um novo **GridContainer**.


## Sinopse

```lua
GridContainer = NodeUI.GridContainer:new(x, y, width, height)
```

## Argumentos
- **`number` x** <br>
Posição horizontal.
- **`number` y** <br>
Posição vertical.
- **`number` width** <br>
Comprimento em pixels.
- **`number` height** <br>
Altura em pixels.

## Retornos
- [`NodeUI.GridContainer`](../node_ui_grid_container/node_ui_grid_container.md) **GridContainer** <br>
Novo **GridContainer**.
