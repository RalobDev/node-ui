# NodeUI.GridContainer

O **GridContainer** organiza seus filhos em uma grade.
 
## Descrição
 
O **GridContainer** organiza seus filhos em linhas e colunas de acordo com o seu número de colunas, que
pode ser definido através de `GridContainer:setColumns()`.


## Métodos

| Nome | Descrição | Retornos |
| ---- | --------- | -------- |
[connect](node_ui_grid_container_connect.md) | Cria uma conexão em determinado sinal do [Control](../node_ui_control/node_ui_control.md). | `nil`
[disconnect](node_ui_grid_container_disconnect.md) | Remove a conexão de um sinal do [Control](../node_ui_control/node_ui_control.md). | `nil`
[getColumns](node_ui_grid_container_get_columns.md) | Retorna a quantidade de colunas do grid. | `number`
[getSeparation](node_ui_grid_container_get_separation.md) | Retorna a separação entre os filhos. | `number`
[new](node_ui_grid_container_new.md) | Cria um novo **GridContainer**. | [`NodeUI.GridContainer`](../node_ui_grid_container/node_ui_grid_container.md)
[setColumns](node_ui_grid_container_set_columns.md) | Define a quantidade de colunas do grid. | `nil`
[setSeparation](node_ui_grid_container_set_separation.md) | Define a separação entre os filhos. | `nil`
